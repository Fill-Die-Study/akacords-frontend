# akacords-frontend

> Akacords is study service (like Inflearn, Class101)

## Getting Started
### Install Dependencies
```bash
$ yarn
```

### Build packages
```bash
$ yarn build
```

## Project Structure
- This is [Monorepo](https://github.com/Fill-Die-Study/akacords-frontend/tree/main/packages) using nx
- `packages/`
  - `admin/`
    - The Admin page
  - `service/`
    - The Service page for user
